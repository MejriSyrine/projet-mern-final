# test_projet (MERN Food App)

MERN stack food/recipes application (Express, MongoDB, React + Vite).

Quick start:

1. Copy `.env.example` to `.env` and fill `MONGO_URI` and `SECRET_KEY`.
2. Start backend: `npm run dev` (from project root).
3. Start frontend: `cd frontend/recipe-app && npm run dev`.
